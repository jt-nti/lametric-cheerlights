# lametric-cheerlights

Simple server to subscribe to CheerLights updates via MQTT and push notifications to LaMetric indicator app

## Usage

```
export ACCESS_TOKEN=*****
export APP_PATH=/api/v1/dev/widget/update/com.lametric.*****
npm start
```
